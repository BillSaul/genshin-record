## 功能

记录抽奖次数、大保底还是小保底、五星出货记录、历史UP卡池记录

## 相关信息

- [Vue-CLI](https://cli.vuejs.org/zh/)开发
- UI组件库[element-ui](https://element.eleme.cn/#/zh-CN)
- 后端数据库[leancloud](https://www.leancloud.cn/)
- 插件[Vuex](https://vuex.vuejs.org/zh/)
